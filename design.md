# StartupLens Design System Specification

## Premium Nebula Theme

### Version 4.0

---

# 1. Design Philosophy

## Core Vision

StartupLens is a premium Web3 design and intelligence section designed for authenticating and validating decentralised infrastructure capabilities. Key features include reusable structure, responsive behavior, and production-ready presentation.

Every page should communicate:
* High-density layouts
* Cryptographic security aesthetics
* Micro-interactions
* Contrast boundaries

---

# 2. Design Principles

### Clean Grid Bounds
A layout emphasizing sharp boundary lines, high-contrast text layout blocks, and medium gray borders (`#C0C0C0`) keeping information grids clean.

### Surface Integration
Components and analytics sections leverage a light gray surface card background (`#F0F0F0`) on a clean white page backdrop (`#FFFFFF`).

### High-Contrast Visual Focus
Core interactive modules (like the pricing component) use dark surface containers to create strong visual highlights and replicate premium SaaS designs.
